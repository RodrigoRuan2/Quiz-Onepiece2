🏴‍☠️ Quiz One Piece

Teste seus conhecimentos sobre o universo de One Piece com esse quiz interativo feito em React!

🚀 Funcionalidades

✅ Perguntas sobre personagens, arcos e curiosidades de One Piece
✅ Pontuação em tempo real
✅ Feedback visual para respostas certas e erradas
✅ Tela de resultado ao final do quiz
✅ Interface responsiva


🛠️ Tecnologias utilizadas

React — biblioteca principal para construção da interface
JavaScript (ES6+) — lógica da aplicação
CSS3 — estilização


📝 Aprendizados
Este projeto foi desenvolvido para praticar:

Componentização no React
Gerenciamento de estado com useState
Renderização condicional
Passagem de props entre componentes
Organização de arquivos em um projeto React
